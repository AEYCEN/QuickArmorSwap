# 📋 Requirements

* **Windows 10 or Windows 11**
* **[Python 3.10+](https://www.python.org/downloads/)** — Make sure to check **"Tcl/Tk, turtle and IDLE"** during installation (it is checked by default)
* **ARK: Survival Evolved** and/or **ARK: Survival Ascended** — Must be installed and launched at least once so the config files exist
